Cebek Relay
===========

#ENG

Script in C# to activate the two relay outputs, wait one second and deactivate the two outputs.
   
## Getting Started
This script is developed in C#. Only you need compile it and use it.

#SPA

## Comenzar a usarlo
Este script está desarrollado en C#. Solamente lo compilas y lo usas.
